Abstract

v3.7 extends the CKP ontology with the agent kernel type (CK.Claude), five formal edge predicates (COMPOSES, TRIGGERS, PRODUCES, EXTENDS, LOOPS_WITH), 20 compliance check types, a four-layer consensus governance engine, task execution via headless Claude Code, dynamic kernel spawning, multi-user NATS sessions, and persona-based LLM capability mounting. All classes maintain BFO 2020 grounding through the four-layer import chain.
